As linguagens que dão vida a este site

Todo site é construído como uma casa: cada parte tem sua função. Aqui usamos três linguagens principais que trabalham em conjunto para criar tudo o que você vê e interage.

HTML (HyperText Markup Language)

O HTML é a estrutura da página — ele define o que aparece no site: textos, imagens, botões, links e seções. Pense nele como o “esqueleto” da construção, organizando todo o conteúdo para que o navegador saiba o que mostrar.

CSS (Cascading Style Sheets)

O CSS é o responsável pela aparência. Ele define cores, tamanhos, fontes e posicionamentos. É o “estilo” da casa — quem escolhe as tintas, os móveis e o jeito que tudo se encaixa visualmente. Sem o CSS, o site até funcionaria, mas seria sem graça e desorganizado.

JavaScript

O JavaScript é o cérebro da operação. Ele traz interatividade, animações e respostas às ações do usuário. É o que faz botões funcionarem, formulários validarem informações e menus se moverem com fluidez. É o que dá vida e dinamismo à página.
